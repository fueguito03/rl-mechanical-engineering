# RL Engineer

## About

This repository documents my journey of learning and researching Reinforcement Learning from first principles.

The goal of this project is not simply to use existing reinforcement learning algorithms, but to deeply understand why they were developed, how they work, and eventually contribute new ideas to the field.

---

## Learning Philosophy

Every algorithm is implemented from scratch whenever possible.

The focus is always:

- Why does this algorithm exist?
- What limitation does it solve?
- What assumptions does it make?
- When does it fail?

Implementation is only one part of learning.
Understanding and questioning every design choice is the primary goal.
